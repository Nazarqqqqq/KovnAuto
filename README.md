# KovAuto

KovAuto - веб-сайт для купівлі та продажу вживаних автомобілів. Проєкт має головну сторінку, каталог авто, сторінку детального перегляду автомобіля, реєстрацію користувачів, особистий кабінет, обране та внутрішні повідомлення.

## Можливості

- реєстрація, авторизація та вихід з облікового запису;
- додавання, редагування та видалення власних оголошень;
- завантаження фотографій автомобілів;
- пошук автомобілів за маркою, моделлю, роком, ціною та містом;
- сортування оголошень за ціною, роком і пробігом;
- додавання автомобілів до обраного;
- позначення автомобіля як проданого;
- перегляд детальної інформації про автомобіль;
- підрахунок переглядів оголошення;
- внутрішні повідомлення між користувачами.

## Технології

- HTML;
- CSS;
- JavaScript;
- Node.js;
- Express.js;
- MySQL;
- XAMPP / phpMyAdmin.

## Запуск проєкту

1. Запустити XAMPP Control Panel.
2. Увімкнути модуль MySQL.
3. Встановити залежності командою:

```bash
npm install
```

4. Створити файл `.env` на основі `.env.example` і вказати дані для підключення до MySQL.
5. Запустити сервер:

```bash
npm start
```

6. Відкрити сайт у браузері:

```text
http://localhost:3000
```

## База даних

Структура бази даних знаходиться у файлі `database/mysql-schema.sql`. Таблиці використовуються для збереження користувачів, автомобілів, фотографій, обраних оголошень та повідомлень.

## Важливо

Файл `.env` не потрібно завантажувати на GitHub, оскільки він містить локальні налаштування та пароль до бази даних. Папку `node_modules` також не потрібно завантажувати, бо її можна відновити командою `npm install`.
